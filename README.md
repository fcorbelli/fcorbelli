# 🧠 Compression Research

Focus areas:

- Incremental archival formats  
- Efficient compression systems  
- Immutable backup strategies  
- Ransomware-resistant backups  
- Long-term data preservation  

Many experiments are built on **ZPAQ**, ensuring **safety, append-only backups, and cloud compatibility**.

---

# 🛡 Ransomware-Resistant Backup Philosophy

- **Append-only writes** → previous backups remain untouched  
- **Incremental transfer** → only changes copied for efficiency  
- **Immutable storage** → all versions kept forever  
- **Cross-platform GUI** → catpaq allows easy interaction  

**Principle:** *Never overwrite, always preserve* ✅
